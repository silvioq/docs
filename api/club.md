Club
====================

Endpoints
---------

### GET /contest_sale_points

Obtiene la cantidad de puntos otorgados por dinero en ventas por un club.

No requiere parametros adicionales.


#### GET /contest_sale_points

`HTTP/1.1 200 OK`

```json
{
  "status":true,
  "data": {
    "contest_sale_points": 5
  }
}
```
